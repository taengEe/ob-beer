SINCE 1933

오비맥주는 1933년 태동해 지난 90여년간 한국 주류산업을 이끌어 온 대한민국 대표 ‘맥주 전문기업’ 입니다.

오비맥주는 국내 맥주 수출 1위이며 한국 맥주 세계화에도 앞장서고 있습니다.

Better future

오비맥주는 현재에 안주하지 않고 더 나은 브랜드 경험을 위해 노력합니다.

VIEW MORE

brand

CASS Fresh

1994년 출시한 오비맥주 대표 맥주 ‘카스’는 끊임없는 혁신과 탁월한 제품력으로 한국 맥주 시장에서 선도적 지위를 유지하고 있습니다.

도수 : 4.5% 종류 : Lager (라거)
Web Site
HANMAC

100% 고품질 국내산 쌀을 첨가하여 한맥만의 상쾌한 풍미를완성시켰으며, 자랑스러운 우리의 ‘K-라거’로써 ‘대한민국 대표라거프로젝트’를 완성해 줄 것으로 기대합니다.

도수 : 4.6% 종류 : Lager (라거)
Web Site
STELLA

벨기에를 대표하는 프리미엄 맥주로 체코산 노블 사츠홉을 사용하여 기분 좋은 쌉쌀한 맛과 청량한 끝 맛을 선사하며, 모든 요리와 환상적인 조화를 이룹니다.

도수 : 5.0% 종류 : Pilsner Lager (필스너 라거)
Web Site
BUDWEISER

버드와이저는 단일 브랜드 판매량 세계 1위의 아메리칸 스타일의 프리미엄 맥주로, 비치우드 숙성공법을 사용하여 깨끗하고 부드러운 맛이 특징입니다.

도수 : 5.0% 종류 : Lager (라거)
Web Site
FIL GOOD

필굿은 시원하고 상쾌한 아로마 홉과 감미로운 크리스탈 몰트를사용해 맛의 품격과 깊이를 더한 하이퀄리티 발포주입니다.

도수 : 4.5% & 7.0% 종류 : 기타 주류
Web Site
HOEGAARDEN

호가든은 독일의 ‘맥주순수령’과는 달리 약초, 허브, 과일 등을 사용해 다양한 맛의 맥주를 개발한 벨기에 사람들의 창의성과 오랜 정통 기법의 만남으로 탄생한 맥주입니다.

도수 : 4.9%, (생맥주 4.8%) 종류 : Belgian White Ale (밀맥주)
Web Site

ESG, GMT

ESG

오비맥주는 환경경영, 사회적 책임경영, 준법·투명 경영을적극 실천하며 기업과 사회가 공존하는 지속가능한 미래를만들기 위해 노력하고 있습니다.

VIEW MORE

GMT

우리는 훌륭한 예비 리더를 성장시키는 것의 중요성을 알고 있습니다. Global Management Trainee Program은 글로벌 핵심 인력으로 성장할 인재를 선발하여 빠른 기간 내에 임원으로 성장시키는 것을 목표로 하는 Fast Track 입니다.
